================================
⚡ 3단계로 아이패드 단독 실행!
================================

## 🎯 초간단 요약

```
1. GitHub에 파일 업로드 (5분)
2. Render.com에 배포 (5분)
3. 아이패드에서 접속 (1분)

총 11분이면 완료!
```

---

## 📦 1단계: GitHub 업로드

### 필요한 파일 5개:
```
✅ server.py
✅ portfolio.html
✅ requirements.txt
✅ Procfile
✅ render.yaml
```

### 업로드 방법:
```
1. github.com 접속
2. 회원가입 (무료)
3. "New repository" 클릭
4. 이름: portfolio-server
5. Public 선택
6. Create 클릭
7. 파일 5개 드래그&드롭
8. Commit!
```

---

## ☁️ 2단계: Render 배포

```
1. render.com 접속
2. "Sign up with GitHub"
3. "New +" → "Web Service"
4. portfolio-server 선택
5. 설정:
   Name: portfolio-server
   Instance Type: Free
6. "Create Web Service" 클릭
7. 3-5분 대기
8. URL 복사!
   예: https://portfolio-server-xxxx.onrender.com
```

---

## 📱 3단계: 아이패드 접속

```
Safari 열기
→ URL/portfolio.html 입력
→ 완료!

예시:
https://portfolio-server-xxxx.onrender.com/portfolio.html
```

### 홈 화면 추가:
```
공유 버튼
→ "홈 화면에 추가"
→ 이름: "포트폴리오"
→ 추가
→ 앱처럼 사용!
```

---

## ⚠️ 주의사항

### 처음 접속 시:
```
20초 정도 기다려주세요
(서버가 깨어나는 중)
```

### 15분 미사용 시:
```
자동 슬립 모드
→ 다시 접속하면 20초 후 작동
```

### 항상 깨어있게 하려면:
```
UptimeRobot.com 사용
→ 무료로 5분마다 핑
→ 슬립 방지
```
